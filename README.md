### Initialize mason

### Adding library
1. Inside your mason.yaml file add
```sh
clean_code:
    git:
      url: https://github.com/ibra-bybuy/clean_architecute_brick.git
```

2. Fetch added library with: mason get

2. Generate folder & file structure using: mason make clean_code
